コンボのダメージは上位数名に入るほど。さらに援軍・必殺技の効果時間中などの条件抜きであれば全キャラ最大ダメージを誇ります。

option.mode内で各種ゲージ位置の調整やダメージ調整（補正・BASARA技のダメージ増加＆＆最大ゲージ本数、ゲージ効率の低下）が設定できます。

■＝＝＝BASARA X入門＝＝＝■
援軍について
Lvが上がるにつれて援軍の使用可能回数や性能が上昇します。
一度援軍を要請するとLvの上昇が極端に遅くなります。
援軍行動使用後援軍が手前で待機している間は再度援軍行動が使用できます。

ゲージ効率（スタイリッシュポイント）は攻撃を受ける・与える・特定の行動をするで上昇します。

スタイリッシュポイントが100%になると画面下の円形状の物体に一撃の文字が光り一撃BASARA発動準備が使用可能となります。そして準備後一定時間内に一撃BASARA技が使用できます。
使用後、準備後一定時間経過後はその試合中一撃BASARA技が使用不能となります（ゲージ効率に変化はなし）。

＝＝＝AIについて＝＝＝
完成度78%です。防御面など課題が結構あります。
一旦攻めが始まるとそのまま封殺することがありますが、逆に懐に入られると何もできずに負けることがあります（割り込み技が使いにくいため）。

以前のAIとは違い相手の行動を記憶して積極性を向上させました（以前のAIは無敵技連発などで楽に倒されないよう攻めてもいい所で待っていたりしていました）。
