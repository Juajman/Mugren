■＝＝＝BASARA X入門＝＝＝■
援軍について
Lvが上がるにつれて援軍の使用可能回数や性能が上昇します。
一度援軍を要請するとLvの上昇が極端に遅くなります。
援軍行動使用後援軍が手前で待機している間は再度援軍行動が使用できます。

ゲージ効率（スタイリッシュポイント）は攻撃を受ける・与える・特定の行動をするで上昇します。

スタイリッシュポイントが100%になると画面下の円形状の物体に一撃の文字が光り一撃BASARA発動準備が使用可能となります。そして準備後一定時間内に一撃BASARA技が使用できます。
使用後、準備後一定時間経過後はその試合中一撃BASARA技が使用不能となります（ゲージ効率に変化はなし）。

＝＝＝AIについて＝＝＝
完成度85%です。防御面など課題が少々ありますが、ステージ・キャラがかみ合えばいろんな所から永久に行きます。

タッグでは性能の関係上中々活躍しづらいと思います（コマ投げ中自分に食らい判定があるので）

＝＝＝仕様など＝＝＝
原作より僅かに援軍ゲージの回復が早いと思います。
これは原作の正確なデータがわからない＋MUGENだと画面がズームしないのでコンボ時間が短くなって原作と同じLVでも同じコンボができない可能性があるためです。
ダメージが低く回復可能体力を大幅に減らすタイプのアシストは攻撃力を1.2倍にしています。

パーフェクトガードはMUGENの仕様上削りKOされたか正確に判別できないようなので削りKOされるようになっております。
誤動作（中段・下段でガードを崩されたときも判別される）前提なら可能ですがこの方法は実装の予定は未定です。

エフェクトの表示量が尋常ではないのでmugen.cfgのExplodMAXは大きめに設定することを推奨します。
ヘルパーの数も多めなので、HelperMAXも増やしておくとよいでしょう。

〜気絶〜
両者、攻撃を当てるたびに相手の気絶値を増やしていき一定値に達すると気絶します。
気絶中はレバガチャ＋ボタン連打で気絶時間を短縮できます。気絶値は一定時間攻撃を食らわない場合徐々に減少していきます。
１コンボ中で増加させる気絶値は制限されているので気絶を狙うときはコンボを途中で止める必要があります。
また、気絶は１コンボ中一回のみ可能です。これは金剛破滅も含まれます。
相手がsimulの場合誤動作の恐れが高いため両者気絶しないようになっております。
