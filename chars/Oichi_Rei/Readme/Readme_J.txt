キャラ固有ゲージを持っておりその状態ではかなりの強さを発揮することが可能。ただし地上戦が弱めで対空も乏しい上に被ダメージが大きいので防御面が厳しい。
コンボのダメージは上位数名に入るほど。さらに瞬間的爆発力は全キャラ最大を誇ります。
弱点はやや座高が大きいこと、広いステージでは本領を発揮できない（画面端に追い詰めることが厳しいため）ことです。

■■＝＝状態異常について＝＝■■

状態異常は以下の技を当てると付きます。
・護れ根の主・陽
・護れ根の主・陰
・護れ根の主・溜め攻撃
・開け根の国中の各種攻撃
・集え根の夢

また、以下の９種類の内上から順番につきます。
・魔：相手の体力時間経過で減少
・攻1/2：相手の攻撃力1/4(タッグは一人に付き1/2)
・防1/2：相手の防御力1/2(タッグは一人に付き1/1.5)
・援軍要請禁止：相手の援軍使用禁止
　※上記非対応の場合（毒：相手のゲージ時間経過で減少）
・気絶耐久値1/2：相手の気絶耐久値半分に減少
・ガード削り：通常技に削り効果
・ガードゲージ1/2：相手のガード耐久値半分に減少
・ＨＰ吸収：相手の体力時間経過で減少し自分の体力時間経過で増加
・バサラゲージ1/2：相手のゲージ増加量半分に減少

■＝＝＝BASARA X入門＝＝＝■
援軍について
Lvが上がるにつれて援軍の使用可能回数や性能が上昇します。
一度援軍を要請するとLvの上昇が極端に遅くなります。
援軍行動使用後援軍が手前で待機している間は再度援軍行動が使用できます。

ゲージ効率（スタイリッシュポイント）は攻撃を受ける・与える・特定の行動をするで上昇します。

スタイリッシュポイントが100%になると画面下の円形状の物体に一撃の文字が光り一撃BASARA発動準備が使用可能となります。そして準備後一定時間内に一撃BASARA技が使用できます。
使用後、準備後一定時間経過後はその試合中一撃BASARA技が使用不能となります（ゲージ効率に変化はなし）。

＝＝＝AIについて＝＝＝
完成度90%です。防御面など課題が結構あります。
一旦開国モード中の攻めが始まるとそのまま封殺することがありますが、逆に迎撃が上手い相手には何もできずに負けることがあります。


＝＝＝仕様など＝＝＝
ダメージが低く回復可能体力を大幅に減らすタイプのアシストは攻撃力を1.2倍にしています。
援軍ゲージの回復速度が原作に比べて若干早いです(30F程度)。

パーフェクトガードは削りKOされたか判別できないようなので削りKOされるようになっております。
誤動作（中段・下段でガードを崩されたときも判別される）前提なら可能ですがこの方法は実装の予定は未定です。

エフェクトの表示量が尋常ではないのでmugen.cfgのHelperMAX,ExplodMAXは大きめに設定することを推奨します。

ヒット数表示がおかしくなる場合があります。常時targetを使っているため起きる不具合で、ある程度は問題なく表示されるようにしてありますが、完璧に直すのは困難です。
